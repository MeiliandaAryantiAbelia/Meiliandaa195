      * Ciri khas kalo website dengan CMS wordpress:
    1. 1. Dibelakang domain ditambah wp-admin, misal https://www.sekawanmedia.co.id/wp-admin/
    2. 2. Dibelakang domain ditambah readme.html
    3. 3. Ctrl U websitenya cari wp-content/uploads/tahun/bulan/namafile
        4. * Cara menginstall Wordpress: 1. Download wordpress di Wordpress.Org di chrome atau pakai file yang dikirim di telegram. 2. Kemudian exstract filenya tersebut. 3. Rename name filenya dan coppy ke xampp htdocsnya, jangan lupa mysqlnya dirunning. Kemudian langsung ke akses localhost/nama folder(localhost/si5a2122o). 4. Pilih Bahasa Indonesia. 5. Kemudian siapkan databasenya dengan membuka xampp lalu klik admin di mysql. 6. Mengisi keterangan koneksi basis data dan mengisi informasi yang dibutuhkan. 7. Klik install Wordpress. 8. Setelah berhasil coba login.
          * Cara backup :
        1. 1. Membuat satu folder terlebih dahulu
        2. 2. Kemudian copy file yang dihtdocs tadi kedalam folder yang dibuat tadi (baru aplikasinya). Kemudian dari phpmyadmin klik export pilih custom pilih database klik go. Taruh file nya di folder tadi
